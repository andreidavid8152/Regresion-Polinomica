def gradient_descent_quadratic(X, y, learning_rate=0.01, iterations=10000, tolerance=1e-6):

    m = len(y)
    theta = np.zeros(X.shape[1])
    cost_history = []
    
    for i in range(iterations):
        # Predicciones
        predictions = X.dot(theta)
        
        # Errores
        error = predictions - y
        
        # Costo (MSE)
        cost = (1/(2*m)) * np.sum(error**2)
        cost_history.append(cost)
        
        # Gradiente
        gradient = (1/m) * X.T.dot(error)
        
        # Parametros
        theta = theta - learning_rate * gradient
        
        # Verificar convergencia
        if i > 0 and abs(cost_history[-2] - cost_history[-1]) < tolerance:
            print(f"Converged at iteration {i}")
            break
    
    return theta, cost_history

# Run gradient descent

# Configuracion y entrenamiento
mlp_configs = [
    {'name': 'Simple MLP', 'hidden_layers': (10,), 'max_iter': 2000},
    {'name': 'Moderate MLP', 'hidden_layers': (50,), 'max_iter': 2000},
    {'name': 'Complex MLP', 'hidden_layers': (1000,), 'max_iter': 2000}
]

mlp_models = {}
mlp_predictions = {}
mlp_metrics = {}

for config in mlp_configs:
    print(f"\nTraining {config['name']} with {config['hidden_layers']} hidden nodes...")
    
    # Creacion MLP
    mlp = MLPRegressor(
        hidden_layer_sizes=config['hidden_layers'],
        activation='relu',
        solver='adam',
        max_iter=config['max_iter'],
        random_state=42,
        early_stopping=False,
        learning_rate_init=0.001,
        verbose=False
    )
    
    mlp.fit(X_scaled, y_scaled)
    
    # Predicciones
    y_pred_mlp_scaled = mlp.predict(X_scaled)
    y_pred_mlp = scaler_y.inverse_transform(y_pred_mlp_scaled.reshape(-1, 1)).flatten()
    
    # Metricas
    mse_mlp = mean_squared_error(y, y_pred_mlp)
    r2_mlp = r2_score(y, y_pred_mlp)
    
    # Guardar resultados
    mlp_models[config['name']] = mlp
    mlp_predictions[config['name']] = y_pred_mlp
    mlp_metrics[config['name']] = {'MSE': mse_mlp, 'R2': r2_mlp}
    
    print(f"  MSE: {mse_mlp:.6f}")
    print(f"  R-squared: {r2_mlp:.6f}")
    print(f"  Training iterations: {mlp.n_iter_}")

Para detectar el sobreajuste, observamos:

6. conclusiones

Desempeo de los modelos:

  • La solucin analtica ofrece un ajuste cuadrtico preciso y eficiente mediante optimizacin matemtica. Sirve como lnea base.
  • El descenso de gradiente converge a coeficientes similares a la solucin analtica, validando la implementacin personalizada.
  • El MLP simple (10 nodos) tiende a producir un ajuste ms parecido al lineal, sin capacidad suficiente para capturar completamente relaciones cuadrticas.
  • El MLP moderado (50 nodos) logra buen desempeo, aproximando con xito la tendencia cuadrtica.
  • El MLP complejo (1000 nodos) puede mostrar el MSE de entrenamiento ms bajo pero corre el riesgo de sobreajustar al memorizar ruido en lugar de aprender el patrn real.

Observaciones de sobreajuste: El MLP complejo con 1000 nodos ocultos muestra caractersticas clsicas de sobreajuste:

  • Error de entrenamiento muy bajo
  • Predicciones potencialmente irregulares que siguen el ruido de los datos
  • Alta complejidad del modelo para una relacin cuadrtica relativamente simple

Mejor modelo: Para esta tarea de regresin cuadrtica, la solucin analtica y el MLP moderado ofrecen el mejor equilibrio entre exactitud y generalizacin. La solucin analtica se prefiere por su eficiencia computacional e interpretabilidad, mientras que el MLP moderado demuestra que las redes neuronales pueden aprender patrones cuadrticos de forma efectiva sin complejidad excesiva.

Compromisos (trade-offs):

  • Analtica: Rpida, precisa, requiere conocimiento del dominio (grado del polinomio)
  • Descenso de gradiente: Educativa, personalizable, requiere ajuste de hiperparmetros
  • MLP: Flexible, puede aprender patrones complejos, propenso al sobreajuste con capacidad excesiva
